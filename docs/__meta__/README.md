# Meta Documentation

This directory contains authoring guides and format specifications for different kinds of project documentation:

- **[features.md](features.md)** — Format specification for feature documentation in `docs/features/`
- **[code.md](code.md)** — Format specification for code guideline documentation in `docs/code/`
- **[code-principle.md](code-principle.md)** — Structure template for principle guideline documents
